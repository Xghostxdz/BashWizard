# BashWizard

This Bash script is a multi-purpose tool designed for system management, including package installation, link checking, and network management. It aims to simplify common tasks for users.

## Features

- **Update Packages:** Update all installed packages on the system.
- **Install Tools:** Install tools like `curl`, `wget`, `net-tools`, and `dnsutils`.
- **Package Management:** Easily install and remove packages.
- **Check URL Safety:** Verify if URLs are safe and accessible.
- **Check DNS:** Query DNS records for specific domains.
- **Check Ports:** Check the status of open ports on a specified host.
- **Clean Temporary Files:** Clean temporary files from the system.
   ```
## Usage

1. Clone or download this repository:
   ```bash
   git clone https://github.com/Xghostxdz/BashWizard
   cd BashWizard

2. Give execution permissions to the script:
   ```bash
   chmod +x install.sh
   ```

3. Run the script:
   ```bash
   ./install.sh
   ```

4. Follow the instructions in the command prompt.

## Requirements

- A Unix-like operating system that supports Bash.
- Tools like `curl`, `wget`, `net-tools`, and `dnsutils` (these can be installed via the script).

## Log

All activities and errors are logged in the `activity_log.txt` file in the current directory. You can review this file for additional information about the executed activities.
